# Data Table

Import `@billy_mcdowell/shadcn-web-components/data-table` and render `shadcn-data-table`. Assign typed `columns` and `data` JavaScript properties. Use `selectable`, `page-size`, `filter-column`, `loading`, and `row-id-key` to configure behavior. The component emits composed events for sorting, filtering, pagination, selection, and column visibility changes.
